# Persona Evaluation: Prep Kitchen Staff (Julia)

**Persona Name:** Julia (Prep Kitchen Staff)
**Role:** Responsible for high-volume, detailed food preparation (chopping, portioning, batch cooking).
**Key Needs:** Speed, accuracy, clear instructions, efficient workflow management, minimal ambiguity.
**Pain Points:** Overwhelming checklists, unclear prioritization, physical strain, managing multiple simultaneous tasks.

---

**Evaluation Summary:**

Julia needs a system that acts as a digital, highly organized expediter. The interface must be extremely clean, prioritizing actionable steps over data entry. The system needs to support batch processing and visual confirmation of completion.

**Key Features to Prioritize:**

1. **Visual Workflow:** Kanban-style boards or step-by-step checklists that can be checked off instantly.
2. **Batch Management:** Ability to group similar tasks (e.g., "Prep all vegetables for Soup A, B, and C") to reduce context switching.
3. **Time/Effort Estimation:** Quick indicators of how long a task should take to help manage the overall prep timeline.
4. **Ingredient Tracking:** Simple input/output logging (e.g., "Used 5 lbs of carrots").

---

**Actionable Recommendations:**

*   **UX/UI:** Implement a "Focus Mode" that strips away all non-essential information, leaving only the current task, required ingredients, and completion button.
*   **Workflow:** Introduce "Prep Batches" that allow managers to assign a group of related tasks to a single worker for maximum efficiency.
*   **Feedback:** Use audible or highly visible visual cues (e.g., a green light turning on) when a station or task is marked as complete.

---

**Overall Score:** 4/5 (High potential, needs significant UX refinement for physical workflow)
