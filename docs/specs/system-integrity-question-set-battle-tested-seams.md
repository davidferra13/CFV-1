# System Integrity Question Set: Battle-Tested Seams

> **Date:** 2026-04-18
> **Purpose:** Every question here maps to a real bug found during live cross-system auditing, not theoretical scenarios. These are the failure patterns that survived 61 domain-level question sets and only surfaced when tracing data across system boundaries end-to-end.
> **Scope:** Cross-boundary seams between: RPC contracts, auth boundaries, AI pipeline, event FSM, financial ledger, notifications, cache invalidation, and side-effect chains.
> **Method:** Each question describes the exact failure mode, the systems involved, and the fix status. Verdicts: FIXED (patched this session), VERIFIED (inspected and confirmed working), KNOWN-GAP (accepted limitation).
