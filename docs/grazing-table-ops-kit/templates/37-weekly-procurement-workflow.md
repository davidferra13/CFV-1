# Weekly Procurement Workflow

## Weekly Cadence

1. Monday: review event calendar and demand forecast.
2. Tuesday: draft event-specific buy sheets.
3. Wednesday: place supplier orders for long-lead items.
4. Thursday: confirm substitutions and backup vendors.
5. Friday: receive/inspect deliveries and log exceptions.
6. Saturday-Sunday: event execution and replenishment notes.

## Order Controls

- [ ] Compare orders to recipe and package standards.
- [ ] Verify minimum orders and delivery cutoffs.
- [ ] Validate cold-chain items and acceptance temperatures.
- [ ] Capture substitutions and cost impacts.

## Post-Week Review

- [ ] Waste/spoilage review
- [ ] Vendor performance notes
- [ ] Cost variance notes
- [ ] Next-week adjustments
