# Client Package One-Sheet

## Signature Experience

Luxury grazing table installations for weddings and milestone events.

## Service Tiers

### Drop-Off

- Curated grazing setup delivered ready to place
- Best for smaller events with in-house venue support

### Styled Setup

- Full on-site styling and arrangement
- Labels, serving tools, and visual finishing included

### Staffed Service

- Styled setup plus service attendants during event window
- Live maintenance and refresh during service

## What Is Included

- Premium cheeses, meats, seasonal fruit, breads, accompaniments
- Visual styling elements and layout design
- Allergy and dietary labeling support
- Setup and post-event pickup options

## Optional Add-Ons

- Floral styling
- Premium caviar or specialty items
- Expanded dietary stations
- Extended staffed hours

## Booking Steps

1. Inquiry and discovery call
2. Proposal and package selection
3. Signed agreement + deposit to reserve date
