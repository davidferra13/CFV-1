# Allergen Control SOP (Template)

## Purpose

Reduce allergen cross-contact risk and ensure accurate guest-facing allergen communication.

## Scope

All events, prep spaces, transport, and on-site service.

## Required Inputs

- Client dietary/allergen form complete and signed.
- Supplier ingredient and allergen statements on file.
- Final menu locked with known substitutions.

## Control Steps

1. Build allergen matrix for every menu item.
2. Designate separate allergen-aware prep tools and surfaces.
3. Store allergen-sensitive items in segregated containers.
4. Label all trays with item name and allergen markers.
5. Use dedicated serving utensils for restricted items.
6. Brief all team members before dispatch.

## Labeling Rules

- Every item label must include:
  - Item name
  - Major allergen indicators
  - Dietary tags where applicable
- If uncertain about an ingredient, do not serve until verified.

## On-Site Rules

- Keep allergen-specific utensils on the correct items.
- Replace utensils immediately if mixed or dropped.
- Escalate all guest allergy questions to the team lead.
- Team members must never guess ingredients.

## Incident Response

1. Remove suspect item from service immediately.
2. Notify client/venue contact.
3. Document incident and corrective action.
4. Preserve packaging/ingredient records for review.
