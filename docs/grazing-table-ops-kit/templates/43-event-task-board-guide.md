# Event Task Board Guide

Use with `spreadsheets/event-task-board.csv` generated per event packet.

## Purpose

Track every critical event milestone from booking through closeout.

## Required Fields

- `due_date`: execution date for the task.
- `phase`: sales, planning, procurement, prep, event-day, closeout.
- `owner`: accountable role.
- `status`: pending, in_progress, completed, blocked.

## Daily Use

1. Review tasks due in the next 3 days.
2. Mark each task status by end of day.
3. Add blocker notes immediately.
4. Escalate blocked tasks that impact event readiness.
