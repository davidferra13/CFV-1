# Inquiry-to-Booking SOP

## Purpose

Convert qualified inquiries into booked events with consistent response speed and process discipline.

## Workflow

1. Inquiry received
2. Lead qualified
3. Discovery call completed
4. Proposal sent
5. Follow-up sequence executed
6. Agreement signed + deposit paid
7. Event moved to production pipeline

## Response SLAs

- New inquiry response: within 4 business hours
- Proposal turnaround: within 48 hours of discovery
- Follow-up cadence: 48 hours, 5 days, final hold notice

## Qualification Criteria

- Event date and venue feasible
- Guest count within service range
- Budget within package range
- Required dietary/allergy scope operationally feasible

## Exit Rules

- If non-responsive after final hold notice, mark as dormant.
- If out of scope, issue polite decline and referral if available.
