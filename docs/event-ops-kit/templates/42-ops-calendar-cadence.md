# Ops Calendar Cadence

## Daily

- Confirm next 72-hour event timeline
- Review inventory exceptions
- Confirm staffing and call times

## Weekly

- Run procurement workflow
- Reconcile event costs vs quotes
- Review KPI dashboard
- Review lead and booking pipeline

## Monthly

- Run P&L review
- Run permit/insurance renewal review
- Run SOP update review
- Run supplier performance review

## Quarterly

- Update package pricing matrix
- Update target margin assumptions
- Refresh venue partner outreach plan
- Review incident trends and retraining needs
