# Cancellation and Reschedule Matrix

## Standard Policy Matrix

| Timing Before Event | Cancellation Fee         | Reschedule Fee          | Notes                                    |
| ------------------- | ------------------------ | ----------------------- | ---------------------------------------- |
| 60+ days            | Deposit retained         | No fee (date available) | New date must be booked within 12 months |
| 30-59 days          | 50% of contracted total  | 10% admin fee           | Subject to vendor commitments            |
| 14-29 days          | 75% of contracted total  | 15% admin fee           | Rush sourcing may apply                  |
| 0-13 days           | 100% of contracted total | Case-by-case            | Product and labor already committed      |
