## PRODUCTION NOTES

- Bakery chefs care deeply about precision. Recipe costing should show exact gram measurements and per-unit costs.
- Seed a recipe with realistic bakery ingredients and measurements (almond flour by gram, not cup).
- Custom order inquiry pipeline should have bakery-specific entries. If "Wedding cake" and "Macaron tower" aren't seeded as inquiry types, create them.
- Price comparison for custom cakes is a powerful sell. If pricing benchmarks don't cover bakery items, show a simulated range.
- Client preference section should include bakery-specific notes: fondant vs buttercream, flavor profiles, design style.
