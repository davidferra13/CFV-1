## PRODUCTION NOTES

- Need a rich client profile with: 12+ events, detailed dietary preferences, kitchen profile with quirky details, occasion dates, loyalty at Gold tier, $28k+ lifetime spend.
- The kitchen profile details (dog name, parking, gate) are what make this memorable. Include specific, realistic details.
- Event history notes should sound like real chef notes: informal, specific, actionable.
- Client list should have 15+ clients across multiple tiers for the filtering/sorting scene.
