**Version**: 1.0
**Date**: 2026-02-14

## Required Fields (Cannot Submit Without)

### 1. Name
- **Why Required**: Must know who is inquiring
- **Min Length**: 1 character
- **Max Length**: 100 characters

### 2. Email
- **Why Required**: Primary contact method
- **Validation**: Must be valid email format

### 3. Message
- **Why Required**: Must explain inquiry purpose
- **Min Length**: 10 characters (ensures meaningful content)
- **Max Length**: 1000 characters

---

**Total Required Fields**: 3
**Total Optional Fields**: 4 (phone, eventDate, guestCount, website/honeypot)
