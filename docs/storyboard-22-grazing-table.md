## A-HA MOMENTS (3 total)

1. **Money:** Vendor costs tracked per item, factored into the margin calculation. 46% after all vendor costs.
2. **Delight:** Portfolio photos embedded in the proposal. The proposal sells the art, not just the price.
3. **Scale:** Event packet with vendor delivery schedules, setup timeline, and packing list. Everything organized for execution.

## PRODUCTION NOTES

- Grazing table specialists are a growing niche. This video should feel aspirational: beautiful food, beautiful business.
- Vendor management features are key. If vendor tracking isn't a dedicated feature, show it as part of the packing list or event notes.
- Portfolio photos in the proposal are a powerful differentiator. If proposals don't support embedded images, note this as a feature to build before recording.
- Event packet/run-of-show may be a document template or a page. If not built as a dedicated feature, show the event detail overview as the "packet."
- Seed the event with realistic grazing table vendors, costs, and a detailed packing list.
- This archetype video should have a slightly more artistic/elegant feel than the catering video. Warmer, more visual.
