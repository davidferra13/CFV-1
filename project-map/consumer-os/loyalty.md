# Loyalty and Rewards

**What:** Points, tiers, gift cards, and rewards to keep clients coming back.

**Routes:** `/my-rewards` (client-facing), various chef-side loyalty management
**Status:** DONE

## What's Here

- Points system: earned per event, redeemable for discounts/vouchers
- Tier system: Bronze/Silver/Gold/Platinum based on spending/frequency
- Gift cards/vouchers: issue, track, redeem via codes
- Rewards catalog: customizable rewards with redemption
- Client visibility: points balance, tier progress, available rewards
- Auto referral points and guest milestones
- Real-time SSE broadcasts for loyalty events
- Hybrid earn mode, setup presets, reward templates
