# Client Portal

**What:** Authenticated client access to their events. Where clients see their bookings, RSVP, submit dietary info, and leave feedback.

**Routes:** `/my-hub`, `/my-events`, `/my-events/[id]`, `/my-rewards`
**Status:** DONE

## What's Here

- Client event list (upcoming/past)
- Per-event detail: countdown, chef contact, guest list with RSVP, dietary forms, pre-event info (parking, dress code, arrival), document sharing (recipe cards, wine pairings, photos, thank-you notes), photo consent, post-event feedback with ratings
- Loyalty rewards: points balance, tier progress, available rewards with redemption
- Account settings (preferences, notifications)

## Open Items

None. Fully functional.
