# Email

**What:** Gmail API integration for sending, syncing, and threading emails with clients.

**Key files:** `lib/email/`, Gmail integration files
**Status:** DONE

## What's Here

- OAuth 2.0 for Gmail API access
- Send email via Gmail (server action wrapper)
- Auto-sync: new inquiry emails detected and parsed via Gmail watch
- Conversation threading linked to inquiries/events
- Email templates (Resend for transactional)
- "At a glance" snapshot footer in client emails (drives portal adoption)
